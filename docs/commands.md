## Basic commands

## List

Display the list of tasks

```bash
all list
```

## Add

Add new task

```bash
todo add -t First -m "Go for a walk"
```

## Show

Mark a task as done

```bash
todo show <id>
```

## Done

Mark as done a task

```bash
todo done <id>
```

## Remove

Remove a to-do

```bash
todo remove <id>
```

Remove all to-do list

```bash
todo remove -a
```
