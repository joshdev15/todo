## Basic commands

## List

Display the list of tasks

```bash
all list
```

## Add

Add new task

```bash
todo add -title First -text "Go for a walk"
```

## Show

Mark a task as done

```bash
todo show <id>
```

## Done

Mark as done a task

```bash
todo done <id>
```

## Remove

Remove a task

```bash
todo remove <id>
```

## Clean

Remove all tasks

```bash
all clean
```
